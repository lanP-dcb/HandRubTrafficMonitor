在Ubuntu 18.04系统下，如果你希望在系统桌面顶栏显示当前程序的运行时间，你可以使用Qt的QTimer类来定时更新程序的运行时间，并通过系统托盘图标显示在顶栏。

以下是一个简单的示例，演示如何在Qt应用中实现这一功能：

  1. 首先，在你的Qt项目中，确保你已经包含了QSystemTrayIcon和QTimer的头文件。
cpp
#include <QApplication>
#include <QMainWindow>
#include <QSystemTrayIcon>
#include <QTimer>
#include <QMenu>
#include <QAction>
#include <QDateTime>
  1. 创建一个MainWindow类，并添加一个QTimer和一个QSystemTrayIcon成员变量。在构造函数中初始化这些成员变量。
cpp
class MainWindow : public QMainWindow
{
    Q_OBJECT

public:
    MainWindow(QWidget *parent = nullptr)
        : QMainWindow(parent)
    {
        // 初始化成员变量
        trayIcon = new QSystemTrayIcon(this);
        trayMenu = new QMenu(this);
        timer = new QTimer(this);

        // 设置系统托盘图标
        trayIcon->setIcon(QIcon(":/path/to/your/icon.png"));

        // 创建菜单项
        QAction *exitAction = new QAction("Exit", this);
        connect(exitAction, &QAction::triggered, qApp, &QCoreApplication::quit);
        trayMenu->addAction(exitAction);

        // 设置托盘菜单
        trayIcon->setContextMenu(trayMenu);

        // 连接定时器信号和槽函数
        connect(timer, &QTimer::timeout, this, &MainWindow::updateTime);

        // 设置定时器间隔（毫秒）
        timer->start(1000); // 1000毫秒 = 1秒

        // 显示主窗口
        show();
    }

private slots:
    // 每秒更新一次运行时间
    void updateTime()
    {
        // 获取当前时间
        QDateTime currentTime = QDateTime::currentDateTime();

        // 计算运行时间
        qint64 elapsedTime = appStartTime.msecsTo(currentTime);

        // 将毫秒转换为时:分:秒格式
        QTime displayTime = QTime(0, 0).addMSecs(elapsedTime);

        // 在托盘图标上显示运行时间
        trayIcon->setToolTip("Running Time: " + displayTime.toString("hh:mm:ss"));
    }

private:
    QSystemTrayIcon *trayIcon;
    QMenu *trayMenu;
    QTimer *timer;
    QDateTime appStartTime = QDateTime::currentDateTime();
};
  1. 在你的项目中添加一个图标文件（比如icon.png），并在.qrc文件中添加：
xml
<!DOCTYPE RCC><RCC version="1.0">
<qresource>
    <file>path/to/your/icon.png</file>
</qresource>
</RCC>
  1. 在主函数中创建QApplication对象，并创建MainWindow实例：
cpp
int main(int argc, char *argv[])
{
    QApplication a(argc, argv);
    MainWindow w;
    return a.exec();
}

这样，你的Qt应用程序就会在系统桌面顶栏的托盘图标上显示运行时间了。在MainWindow的构造函数中，定时器每隔一秒触发一次，更新托盘图标的提示信息，显示程序的运行时间。同时，你也可以根据需要修改菜单项，添加其他功能。

以下是一个使用AppIndicator在系统桌面顶栏中显示运行时间的示例代码。请确保你的项目中包含了libappindicator3库。

cpp
#include <QApplication>
#include <QMainWindow>
#include <QTimer>
#include <QDateTime>

class MainWindow : public QMainWindow
{
    Q_OBJECT

public:
    MainWindow(QWidget *parent = nullptr)
        : QMainWindow(parent)
    {
        // 创建定时器
        timer = new QTimer(this);

        // 连接定时器信号和槽函数
        connect(timer, &QTimer::timeout, this, &MainWindow::updateTime);

        // 设置定时器间隔（毫秒）

        // 初始化 AppIndicator
        indicator = app_indicator_new("my-indicator", "path/to/your/icon.png", APP_INDICATOR_CATEGORY_APPLICATION_STATUS);
        app_indicator_set_status(indicator, APP_INDICATOR_STATUS_ACTIVE);

        // 显示主窗口

    // 每秒更新一次运行时间
    void updateTime()
    {
        // 获取当前时间
        QDateTime currentTime = QDateTime::currentDateTime();

        // 计算运行时间
        qint64 elapsedTime = appStartTime.msecsTo(currentTime);

        // 将毫秒转换为时:分:秒格式

        // 设置 AppIndicator 的标签

    }

private:
    QTimer *timer;
    QDateTime appStartTime = QDateTime::currentDateTime();
    AppIndicator *indicator;
};

int main(int argc, char *argv[])
{
    QApplication a(argc, argv);
    MainWindow w;
    return a.exec();
}

请替换上述代码中的 "path/to/your/icon.png" 为你实际的图标路径。此示例使用了 libappindicator3 库的基本功能，确保你的系统中已经安装了这个库。

在这个示例中，AppIndicator的标签会被设置为实时的运行时间，并显示在系统桌面的顶栏中。
